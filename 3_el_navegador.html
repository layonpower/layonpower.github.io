3: El Navegador

Los diferentes navegadores web, pese a tener la misma función, la pueden desempeñar de diferente manera ya que, dependiendo del navegador, la visualización de la web puede ser distinta debido a que cada navegador puede contener más o menos prestaciones .
El desarrollo de los diferentes navegadores no ha seguido las mismas pautas lo que hace que lo que está escrito en una web pueda ser válido para un determinado navegador, pero no para otro.
No obstante, de forma muy general, las páginas web se interpretan prácticamente igual en todos los navegadores.
