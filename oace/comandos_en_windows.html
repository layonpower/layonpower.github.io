Comandos en Windows

Recopilación de los comandos generales vistos en clase

help

Muestra un listado con todos los comandos que podemos utilizar y lo que hace cada uno de ellos

help

Si queréis ver la ayuda de un comando en concreto deberéis escribir el nombre del comando /?

mkdir /?

(imagen y explicación de corchetes y demás)

cls

Borra el terminal

cls

mkdir

Nos permite crear carpetas. La sintaxis es mkdir nombreCarpeta. Ejemplos:

mkdir documentos

También podemos crear mas de una carpeta a la vez

mkdir fotos videos

El comando anterior crearía 2 carpetas a la vez, una llamada fotos y otra llamada videos

dir

Lista el contenido de una carpeta

dir

cd

Sirve para cambiar al directorio que se facilite como parámetro. Se puede usar una ruta absoluta

cd c:\carpeta

o una relativa

cd ..

cd .\fotos\verano

Cambiar de unidad

El comando cd nos permite cambiar de directorio pero no de unidad. En windows las distintas particiones tienen distintas letras de unidad. Si queremos cambiar a una unidad deberemos de escribir en el terminal el nombre de la unidad seguido de : Vamos a ver a continuaión como cambiar a una unidad llamada J

j:

copy

Sirve para copiar ficheros. La sintaxis es copy origen destino Vamos a ver un ejemplo

copy nominas.xls nominasEnero.xls

Copia el fichero nominas.xls a un nuevo fichero llamado nominasEnero.xls de modo que tendremos los 2 ficheros.

chkdsk

Comprueba un disco un muestra un informe de estado

chkdsk

Puede ser que te pida que ejecutes el comando como administrador. Para ello encima del icono que abre CMD debes hacer botón derecho y elegir la opción que dice Ejecutar como administrador

date

Muestra o establece una fecha. Si consultamos la ayuda del comando con date /? veremos que si ejecutamos

date /t

Nos mostrará la fecha, Mientras que si escribimos únicamente

date

Nos pedirá que pongamos una nueva fecha (si no queremos cambiarla podemos darle a enter y así se queda la fecha actual)

del

Borrar ficheros. La sintaxis es del nombreDelFichero

del fichero.doc

del *.txt

El primer ejemplo borrará un fichero llamado fichero.doc y el segundo ejemplo borrará todos los ficheros que cumplan el patrón de búsqueda (lee el apartado Comodines que aparece a continuación).

echo

Escribe lo que pongamos a continuación

echo Hola mundo

Escribirá por pantalla Hola mundo.

También podemos utilizar el comando echo para crear ficheros o añadirles contenidos. Por ejemplo

echo Hola >> hola.txt

Si el fichero hola.txt existe añade al final del mismo el texto Hola. Si el fichero no existe, lo crea con el texto Hola.

mklink

Simplificando mucho podemos decir que nos sirve para crear accesos directos. Su sintaxis es:

mklink nombreEnlace destino

Si queremos hacer un enlace a un directorio podemos hacerlo, por ejemplo, así:

mklink /d nombre c:\destino

Fíjate en que si entras en el entorno gráfico en la carpeta nombre que acabas de crear la dirección no te muestra que realmente has accedido a c:\destino Sin embargo, si ejecutas el comando cd verás que nombre es un SYMLINKD y verás que su destino es c:\destino

move

Sirve para cambiar la ubicación de un fichero. La sintaxis es move origen destino. Vamos a ver un ejemplo:

move fotoVerano.jpg \backup\fotoVeranoConAldara.jpg

Quita el fichero fotoVerano.jpg de su ubicación actual y lo coloca dentro de la carpeta backup con el nombre fotoVeranoConAldara.jpg

rmdir

Borra directorios

rmdir videos

rename

Cambia el nombre de uno o mas archivos. Su sintaxis es

rename origen destino

Por ejemplo, si no encontramos en un directorio que contiene una fichero llamado mapa.bmp y queremos renombrarlo a foto.bmp deberemos ejecutar

rename mapa.bmp foto.bmp

También sirve para renombrar directorios.

shutdown

Nos permite apagar el equipo. Existen una gran diversidad de opciones para este comando.

tree

Dibuja la jerarquía de directorios de la ruta en que nos encontremos o de la ruta que facilitemos como parámetro.

tree

tree c:\ventas

xcopy

Copia árboles de directorios y archivos. Su sintaxis es

xcopy origen destino

Si queremos copiar una carpeta llamada Paris ubicada dentro de c:\comodines en una carpeta llamada c:\borrame

la sintaxis sería

xcopy c:\comodines\Paris c:\borrame
